PostgreSQL Database Management System
=====================================

* this repo == 
  * source code distribution of the PostgreSQL +
  * C language bindings 

* PostgreSQL
  * == advanced object-relational database management system /
    * supports
      * extended subset of the SQL standard 
        * transactions,
        * foreign keys,
        * subqueries,
        * triggers,
        * user-defined
          * types
          * functions
